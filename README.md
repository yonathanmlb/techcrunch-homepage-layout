This project recreates a TechCrunch-style homepage using HTML and CSS. The goal of the project was to practice building complex editorial layouts using CSS Grid, Flexbox, and responsive design techniques.

## Preview

Open the project with Live Server or open `index.html` directly in your browser.

## Layout Reference

The layout was recreated based on a design inspired by the TechCrunch homepage.

## Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Responsive Design

## Features

* Left sidebar navigation menu
* Featured article section
* Right column trending headlines
* News article cards
* Editorial style layout
* Responsive multi-column design

## What I Practiced

* Building complex multi-column layouts
* Using CSS Grid for page structure
* Combining Grid and Flexbox
* Structuring editorial news-style pages
* Responsive design for desktop and mobile
